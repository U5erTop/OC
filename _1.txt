1. Создание папок
Команда: mkdir Исходные_данные Архив

mkdir (Make Directory) - команда для создания новых папок

Что делает: Создает две новые папки в текущем расположении

Результат:

Папка Исходные_данные (пустая)

Папка Архив (пустая)

2. Создание текстового файла
Команда: echo Главный отчет > отчет.txt

echo - команда вывода текста

> - оператор перенаправления вывода в файл

Что делает: Создает текстовый файл и записывает в него указанный текст

Результат: Файл отчет.txt с содержимым "Главный отчет"

3. Копирование файла
Команда: copy отчет.txt Архив

copy - команда копирования файлов

Что делает: Копирует файл из текущей директории в указанную папку

Результат:

Оригинальный файл отчет.txt остается на месте

Копия файла создается в папке Архив

4. Переименование файла
Команда: ren Архив\отчет.txt бэкап_отчета.txt

ren (rename) - команда переименования

\ - разделитель пути к файлу

Что делает: Переименовывает файл внутри папки Архив

Результат: Файл Архив\отчет.txt переименован в Архив\бэкап_отчета.txt

5. Просмотр содержимого папки
Команда: dir Архив

dir (directory) - команда просмотра содержимого папки

Что делает: Показывает список файлов и подпапок в указанной директории

Результат: Выводит в консоль информацию о файле бэкап_отчета.txt (имя, размер, дата изменения)
